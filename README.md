# Shaik Irfan Basha — Developer Portfolio V2

![Premium Developer Portfolio](https://img.shields.io/badge/Status-Live-success?style=for-the-badge&logo=github)
![Tech Stack](https://img.shields.io/badge/Stack-Vanilla%20JS%20%7C%20CSS3%20%7C%20HTML5-blue?style=for-the-badge)

A visually stunning, data-driven personal portfolio built with a focus on premium aesthetics, performance, and clean code. This project showcases my skills, experience, and the products I've built using a modern dark-themed interface.

## 🌟 Key Features

- **Premium Dark Mode UI**: A curated aesthetic using glassmorphism, subtle gradients, and modern typography.
- **Dynamic Content Injection**: Skills, projects, and contact links are managed in a centralized data layer (`script.js`), making updates effortless without touching HTML.
- **Interactive Experience**: Smooth scroll animations, intersection observers for "reveal" effects, and a responsive mobile drawer.
- **Zero Dependencies**: Built entirely with Vanilla JavaScript, HTML5, and CSS3 for maximum performance and portability.
- **SEO Optimized**: Follows best practices for metadata and semantic HTML.

## 🛠 Tech Stack

- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Animations**: CSS Transitions & Intersection Observer API
- **Icons**: Custom SVG paths for a lightweight footprint

## 🚀 Getting Started

To run this project locally, you have two options:

### Option 1: Live Server (Recommended)
If you have Node.js installed, you can use the built-in script:
```bash
npm start
```
This will launch a local development server.

### Option 2: Direct Open
Simply open `index.html` in any modern web browser.

## 🌐 Deployment

This project is optimized for **GitHub Pages**. To deploy:
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/ (root)` folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.
